## Getting Started

### For Developers

1. **Fork our repository**: Visit [gitlab.com/newnepal/newnepal.org](https://gitlab.com/newnepal/newnepal.org)
2. **Set up locally**:
   ```bash
   git clone [your-fork-url]
   cd newnepal.org
   yarn install
   yarn start
   ```
3. **Important**: All content changes must include Nepali translations
4. **Submit merge requests** for your contributions

### For Non-Developers

- **Content Creation**: Help us write engaging content for our platform
- **Translation**: Assist with English-to-Nepali translations
- **Community Outreach**: Help us connect with more youth groups
- **Feedback**: Share your ideas and suggestions
